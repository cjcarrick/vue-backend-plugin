# vue-backend-plugin

A Vue CLI plugin that adds a backend server, SCSS globals, Typescript, and makes stylistic changes to prettier.

### Recomended usage

Add an alias to your .zshrc:

```sh
alias nv="vue create --preset cjcarrick/vue-backend-plugin"
```
